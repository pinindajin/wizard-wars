# Wizard Wars Agent Guide

## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues for `pinindajin/wizard-wars`. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default five-label triage vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a multi-context layout with root `CONTEXT-MAP.md`, system ADRs in `docs/adr/`, and context docs grouped around web app UI, browser game client, shared game contracts and rules, authoritative simulation, realtime rooms, backend platform, and delivery ops. See `docs/agents/domain.md`.

Repo docs are canonical for active agent/code context. Obsidian remains the home for raw research, session logs, brainstorming, and historical working memory.
